  • __init__(self,position, wide,column, row,r,c, **kwargs

    Permet de spécifier les dimensions de l'encadré contenant une image favorite sur l'interface.

    • Paramètres
      1. event: tkinter.Event: L'événement qui a déclenché l'appel de cette fonction (par défaut, None).
      2. size (int): La taille de l'objet, en pixels.
      3. column (int): La colonne dans laquelle l'objet doit être placé.
      4. row (int): La ligne dans laquelle l'objet doit être placé.
      5. **kwargs (dict): Tout argument supplémentaire.
    • Attributs
      1. wd (int): La largeur de l'objet, en pixels.
      2. ht (int): La hauteur de l'objet, en pixels.
      3. large (int): La taille de l'objet agrandie, en pixels.
      4. photo_image (tkinter.PhotoImage): L'image associée à l'objet.
      5. note (str): Une note associée à l'objet.
      6. id (int): L'identifiant unique de l'objet.
      7. col (int): La colonne dans laquelle l'objet est placé.
      8. row (int): La ligne dans laquelle l'objet est placé.
    • Retourne

      None

  • Make_favorite (self,id, note, image )
