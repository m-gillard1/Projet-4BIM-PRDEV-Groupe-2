Main

Les fonctions concernant le main se trouvent dans le fichier "main_sprint1.py".

Fonctions

  • creation_list_note(nb_image_par_vague)

    Créer une liste de note aléatoire pour permettre de tester nos fonctions sans l'IHM.

    • Paramètres>
      1. nb_image_par_vague : int: donne le nombre d'image dans une vague et donc le nombre de note à générer.
      2. Retourne

        note : list: Liste d'entiers qui serviront de notes.

  • encoded_image (path_im_vague)

    Encode toutes les images de la vague en vecteur grace à l'autoencodeur.

    • Paramètres>
      1. path_im_vague : str : lien vers le dossier contenant les images de la vague en cours.
      2. Retourne

        encoded_image_list : list: liste de vecteurs correspondant aux images encodées (pas de note).

  • data_structure_note_image(encoded_image_list, note)

    Creation de la structure de données qui va etre utilisée pour l'algorithme genetique soit : une list contenant pour chaque image 2 vecteurs de meme taille (taille du vecteur de l'image encodée) un avec la note comme premier élément puis Nan et le 2eme vecteur contenant l'image encodée.

    • Paramètres>
      1. encoded_image_liste : list : liste de vecteurs des images encodées.
      2. note : list de int: list de int contenant les notes dans l'ordre corresponant à encoded_image_list.
      3. Retourne

        image_note_list : list: list contenant pour chaque image une list avec 2 vecteurs : le 1er element du 1er vecteur contient la note puis le reste de ce vecteur contient des Nan (afin d'avoir un vecteur de la meme taille que le vecteur de l'image encodée) et le 2e vecteur correspond à l'image encodée.
